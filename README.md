# TaskFlow Pro - Smart Task Management System

TaskFlow Pro is a web-based task management system developed using the Laravel framework. It helps users create, organize and manage daily tasks in a simple and easy way. The system includes secure user authentication, task management, category management and a clean dashboard to improve productivity.

## Features

- User Registration and Login
- Secure Authentication System
- Dashboard with Task Summary
- Create, Edit, and Delete Tasks
- Organize Tasks with Categories
- Mark Tasks as Completed or Pending
- Search and Filter Tasks
- Responsive and Easy-to-Use Interface

## Installation

1. Clone the repository.

```bash
git clone https://github.com/your-username/TaskFlow-Pro.git
```

2. Open the project folder.

```bash
cd TaskFlow-Pro
```

3. Install Composer packages.

```bash
composer install
```

4. Install Node packages.

```bash
npm install
```

5. Create the environment file.

```bash
copy .env.example .env
```

6. Generate the application key.

```bash
php artisan key:generate
```

7. Configure your MySQL database in the `.env` file.

8. Run database migrations.

```bash
php artisan migrate
```

9. Start the Laravel development server.

```bash
php artisan serve
```

10. Open your browser and visit:

```
http://127.0.0.1:8000
```
